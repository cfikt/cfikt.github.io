HAProxy 简介

HAProxy 介绍

HAProxy 是一款基于 TCP（四层）和 HTTP（七层）的负载均衡代理软件，它是免费、快速并且可靠的一种解决方案。HAProxy 具备一万以上的高并发，支持为后端服务器提供高可用机制，支持基于 Cookie 的持久性、自动故障切换等功能，适用于高负载的环境下使用

​

HAProxy 是一款高性能的开源负载均衡软件，能够实现对多个服务器的负载均衡以及高可用性的支持。它能够在应用层对请求进行转发，支持 TCP 和 HTTP 协议，具有高性能、高可用性、灵活性和可靠性等特点

​

HAProxy 的主要作用是将客户端请求分发到多个后端服务器，并且能够根据不同的负载均衡算法进行分发，以达到平衡负载的目的。同时，HAProxy 还能够进行健康检查，自动排除故障节点，保证服务的高可用性。此外，HAProxy 还支持 SSL 加密、HTTP 压缩、请求限流等功能，能够提高应用的性能和安全性

​

​

HAProxy 功能

HAProxy 优点：

  • 支持 TCP 和 HTTP 反向代理
  • SSL/TSL 服务器
  • 可以针对 HTTP 请求添加 Cookie，进行路由后端服务器
  • 可平衡负载至后端服务器，并支持持久连接
  • 支持所有主服务器故障切换至备用服务器
  • 支持专用端口实现监控服务
  • 支持停止接受新连接请求，而不影响现有连接
  • ……

​

HAProxy 缺点：

  • 不支持正向代理
  • 不支持缓存代理
  • 不支持 Web 服务
  • 不支持 UDP 协议
  • 单机性能与 LVS 相比较差

​

​

​

​

编译安装 HAProxy

HAProxy 支持 RedHat 系统的 yum 安装以及 Ubuntu 操纵系统的 apt 安装，但此种方法安装版本较低，若想安装最新版本，建议从官方网站下载源码包进行编译安装

​

​

解决 Lua 环境

HAProxy 支持基于 Lua1实现功能扩展，由于操作系统自带的 Lua 版本较低，并不符合 HAProxy 要求的 Lua 最低版本（5.3）的需求，因此需要编译安装较新版本的 Lua 环境，然后才能编译安装 HAProxy
