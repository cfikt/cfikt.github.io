TCP 三次握手

假设有两个应用程序 A 和 B，它们需要建立 TCP 连接。以下是三次握手的过程：

  • 第一次握手：A 发送一个 SYN（同步）包给 B，表示 A 要和 B 建立连接
  • 第二次握手：B 收到 A 的 SYN 包后，回复一个 SYN+ACK（同步和确认）包给 A，表示 B 接受 A 的请求，并准备建立连接
  • 第三次握手：A 收到 B 的 SYN+ACK 包后，发送一个 ACK（确认）包给 B，表示 A 接受 B 的回复，并准备建立连接

TCP 四次挥手

假设应用程序 A 和 B 已经建立了 TCP 连接，现在需要断开连接。以下是四次挥手的过程：

  1. 第一次挥手：A 发送一个 FIN（结束）包给 B，表示 A 不再发送数据
  2. 第二次挥手：B 收到 A 的 FIN 包后，发送一个 ACK 包给 A，表示 B 已经收到了 A 的请求
  3. 第三次挥手：B 发送一个 FIN 包给 A，表示 B 不再发送数据。
  4. 第四次挥手：A 收到 B 的 FIN 包后，发送一个 ACK 包给 B，表示 A 已经收到了 B 的请求

四次挥手需要进行两次确认，是为了防止在连接关闭过程中出现数据包丢失或者延迟的情况，从而保证连接关闭的可靠性

什么是 TCP/IP 协议

TCP/IP 协议是一种网络通信协议，它是 Internet 的基础。TCP/IP 协议的主要功能是实现数据的传输和通信，它的特点是可靠性高、传输速度快、灵活性强等

什么是 IP 地址

IP 地址是一种用于标识网络中计算机的地址。IPv4 地址是由 32 位二进制数表示的，而 IPv6 地址是由 128 位二进制数表示的。IPv4 地址的数量有限，已经开始逐渐被 IPv6 地址所取代

什么是子网掩码

子网掩码是一种用于划分 IP 地址的网络地址和主机地址的标识符。它的作用是将 IP 地址划分为网络地址和主机地址两部分，以便于进行网络通信和管理

什么是路由器

路由器是一种用于连接不同网络的设备，它的主要功能是将数据包从一个网络传输到另一个网络。路由器可以根据不同的网络地址和路由表来进行数据包的转发和路由选择

什么是防火墙

防火墙是一种用于保护计算机和网络安全的设备，它的主要功能是控制网络流量，防止未经授权的访问和攻击。防火墙可以根据不同的规则和策略来进行流量过滤和访问控制

什么是 DNS

DNS 是一种用于域名解析的协议，它的作用是将域名转换为 IP 地址

  1. 客户端发起一个 DNS 请求到本地 DNS 服务器
  2. 请求到达本地 DNS 服务器，本地服务器首先查询缓存记录，若缓存中存在此记录，则直接返回结果，若不存在此纪录，则本地服务器向根 DNS 服务器进行迭代查询
  3. 根 DNS 服务器没有记录请求对应的域名和 IP 的对应关系，告诉本地 DNS 服务器，你可以到顶级 DNS 服务器上去查询，并给出顶级 DNS 服务器的地址
  4. 本地 DNS 服务器继续向顶级 DNS 服务器发出请求，顶级 DNS 服务器告诉本地 DNS 服务器所查询的权威服务器的地址
  5. 最后，本地 DNS 服务器向权威服务器进行查询，权威服务器告诉本地 DNS 服务器所查询域名的 IP
  6. 本地服务器将查询到的 IP 返回给客户端，并写入缓存
  7. 若客户端再次发起同个 DNS 请求，则直接从缓存中读取结果

SSL/TLS 协议是一种用于网络通信加密的协议，它的主要作用是保护网络通信的安全性和隐私性。SSL/TLS 协议可以在客户端和服务器之间建立加密通道，以防止数据被窃取或篡改

TCP 与 UDP

TCP 和 UDP 是两种常见的传输层协议：

  • TCP 是一种面向连接的协议，它在通信前要先建立连接，然后进行数据的传输和通信。TCP 协议具有可靠性高、传输速度较慢、数据传输有序等特点。TCP 协议采用三次握手和四次挥手的方式来建立和关闭连接，可以保证数据的可靠传输和完整性
  • UDP 是一种无连接的协议，它在通信前不需要建立连接，直接进行数据的传输和通信。UDP 协议具有传输速度快、数据传输无序、可靠性低等特点。UDP 协议不提供数据的可靠传输和完整性保证，但它适用于一些对数据传输速度要求较高、数据传输可靠性要求不高的应用场景，例如视频和音频流传输

网络七层模型

网络七层模型是指计算机网络体系结构中的七个层次，每个层次负责不同的功能

  • 应用层：负责提供各种网络应用程序
  • 表示层：负责将数据进行编码、压缩和加密等处理，以便于应用程序的解释和使用
  • 会话层：负责建立、管理和终止应用程序之间的会话，包括数据流的同步和恢复等
  • 传输层：负责提供端到端的数据传输服务，包括数据分段、数据重组和流量控制等
  • 网络层：负责在多个数据链路之间进行路由选择和数据包转发，实现不同网络之间的通信
  • 数据链路层：负责在物理层之上建立数据链路，实现数据的可靠传输和错误检测
  • 物理层：负责将数据从一个节点传输到另一个节点，包括物理介质、信号和数据传输速率等
