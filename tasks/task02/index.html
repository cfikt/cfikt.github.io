云计算 SRE 运维 — 第二周

发表于2022-10-09|更新于2023-01-02|云计算 SRE 运维 - 作业集
|字数总计:690|阅读时长:3分钟

完成一个 SHELL 脚本，脚本的作用：

  1. 运行脚本可以显示出本机的 ip 地址
  2. 如果 ip 地址中有 3 这个数字，那么就打印出当前的系统时间
  3. 如果 ip 地址中不含 3 这个数字，就批量建立用户 test_1, test_2, … test_10 并且所有用户同属于 test 组
  4. 打印出 /etc/passwd 这个文件中可以登陆的用户（非 /usr/sbin/nologin）
  5. yum 安装 nginx 服务，并且启动该服务
  6. 一个脚本完成
