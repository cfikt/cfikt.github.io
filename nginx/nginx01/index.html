Nginx 配置文件

配置文件说明

Nginx 配置文件（指令）官方手册：https://nginx.org/en/docs/dirindex.html

Nginx 配置文件组成部分：

  • 主配置文件：nginx.conf
  • 子配置文件：include conf.d/*.conf
  • mime.types：支持的 mime 类型，资源媒体类型（即导入支持的文件类型）
  • 其它文件：其它协议相关的配置文件

​

配置文件格式

Nginx 中配置文件由指令块和指令构成，每条指令以分号结尾，指令与值之间以空格分隔

  • 指令块：以大括号将多条指令组织在一起，且支持嵌套指令块
  • 允许使用 INCLUDE 语句组合多个配置文件，提高可维护性
  • 部分指令支持正则表达式，支持 $ 符号使用变量（内建变量直接引用；自定义变量使用 SET 命令定义）
